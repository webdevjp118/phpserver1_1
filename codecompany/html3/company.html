私たちの多様な事業とコミュニティ主体のビジョン

CODE Groupでは、「資本主義への挑戦。コミュニティが主体となる未来を創る」というグループビジョンを掲げています。私たちの事業は、単にビジネスの多様性を追求するのではなく、コミュニティ主体の価値創造を目指すことに深く根ざしています。農業から漁業、医療、健康食品、EV三輪車、温泉に至るまで、私たちの事業は「どこにもないここだけにあるもの」を生み出すプロセスです。

私たちが目指すのは、単なる製品やサービスの提供にとどまらず、コミュニティの一員としてそれぞれが独自の価値とエネルギーを持ち寄り、新たな創造を生むことです。それぞれの事業分野で得た知識と経験は、コミュニティに活力をもたらし、私たちが望む未来を具体化します。

例えば、農業や漁業では、持続可能な方法での食料生産を通じて、私たちのコミュニティに栄養豊かな食べ物を提供します。医療分野では、革新的な治療法の開発によってコミュニティの健康を守ります。EV三輪車や温泉事業は、環境に配慮しつつ、生活の質を高める手段を提供します。
このように、私たちの事業は、コミュニティが主体となり、それぞれのメンバーが求める「良いもの」や「欲しいもの」を創り出し、享受する循環を生み出しています。CODE株式会社の取り組みは、コミュニティを中心に据えることで、新たな価値とつながりを生み出し、持続可能な未来へと導いています。

私たちの社会活動と地域創生

CODE Groupは、私たちのコミュニティと地域社会が共に繁栄する未来を目指し、多様な社会活動に取り組んでいます。それは地方でしかできないことも多くあるためです。
これらの活動は、私たちの夢を実現するために、地域創生が絶対に必要不可欠な課題であるという信念に基づいています。

海洋深層水を使った養殖・畜養システム

海洋深層水を利用した独自の養殖システムは、水産物の乱獲問題に対する革新的な解決策です。このシステムにより、一年中出荷可能で環境に優しい養殖を実現しています。これは地域の持続可能な資金源となり、安定した供給を可能にします。

循環型畜産モデル

このモデルは、豚が有機野菜の土を耕し、自然な肥料を提供するという循環型の農業アプローチです。さらに、出荷できない野菜の部分や茎、根などの廃棄物も全て豚の餌として利用されます。これにより、無駄を減らし、地域の農業を持続可能なものに変えています。

濃縮温泉

環境保護を考慮し、地方の生活をサポートするEV三輪は、維持費を抑えつつ、車に比べて安価で便利な移動手段を提供しています。これは地方の新たな移動のスタンダードとして、観光事業にも最適な選択肢です。

EVマイクロモビリティ

これらの取り組みを通じて、CODE Groupは、地域の経済活性化と環境保全を同時に実現し、地域社会の豊かさを追求しています。私たちの地域創生の活動は、自分たちのコミュニティのために多様な事業を展開することと、地域に新たな価値をもたらすという、私たちのビジョンを具現化しています。
